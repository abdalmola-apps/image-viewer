# Image Viewer
An addon to preview image without download in [Odoo](https://www.odoo.com) open source
    
# Depends
  * [Fancybox](http://fancybox.net) - Fancy jQuery lightbox alternative
    
# How to use
  * Install addon [Settings > Modules > Install Modules]
